# UML

## 图

diagram

一组元素的图形表示，大多数情况下把图画成顶点和弧的连通图。
为了对系统进行可视化，可以从不同的角度画图，这样图是对系统的投影。

UML2.0提供了13种图，分别是 类图、对象图、用例图、序列图、通信图、状态图、活动图、构件图、组合结构图*、部署图、包图*、交互概览图*、计时图*。
